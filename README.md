# BLE Scanner Flutter App

This project is a Flutter application that allows users to scan for and connect to Bluetooth Low Energy (BLE) devices. It provides a simple and user-friendly interface to discover nearby BLE devices, view their details, and establish connections.

## Features

- **Scan for BLE Devices:** Easily search for nearby Bluetooth Low Energy devices.
- **Device List:** View a list of discovered devices with their names, IDs, and signal strength (RSSI).
- **Connect to Devices:** Tap on a device to initiate a connection.
- **Modern UI:** Clean and responsive design using Flutter's Material components.
- **State Management:** Efficient state handling using the GetX package.

## Getting Started

1. **Clone the repository:**
   ```sh
   git clone https://github.com/survesufiyan/ble_scanner1.git
   cd ble_scanner1
   ```

2. **Install dependencies:**
   ```sh
   flutter pub get
   ```

3. **Run the app:**
   ```sh
   flutter run
   ```

## Packages Used

- [`flutter_blue_plus`](https://pub.dev/packages/flutter_blue_plus) - BLE scanning and connection
- [`get`](https://pub.dev/packages/get) - State management
